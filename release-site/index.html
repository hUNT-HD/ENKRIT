Install notes

macOS — builds are unsigned. On first launch: right-click the app → Open, or System Settings → Privacy & Security → "Open Anyway".

Windows — extract the ZIP and run ENKRIT.exe. If SmartScreen warns, click "More info" → "Run anyway".

Android — enable "Install unknown apps" for your browser, then open the APK.
